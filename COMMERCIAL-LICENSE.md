## License Validation

Commercial licenses are validated through a simple license key system:

1. Purchase a license at [graphdb.dev/pricing](https://graphdb.dev/pricing) (coming soon)
2. Receive your license key via email
3. Set the `CLUSO_LICENSE_KEY` environment variable
4. The database validates the key on startup

**No features are locked.** The Community Edition is fully functional.
